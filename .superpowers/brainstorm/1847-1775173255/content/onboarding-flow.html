Transition Summary

Welcome → Scanning: Content cross-fades (SwiftUI .transition(.opacity)). Sidebar expands via NSSplitViewItem.isCollapsed = false (AppKit native animation).
Scanning → Launcher: Content cross-fades when scan completes. Sidebar already visible, repos already populated.
No "Add Folder" on launcher: Toolbar button (⌘⌥⇧O) handles subsequent folder additions.
